## Box Python SDK v2.10.0 released

  **New Features and Enhancements:**

- Add support for `copyInstanceOnItemCopy` field for metadata templates ([#546][2020-10-02_1])
- Allow creating tasks with the `action` and `completion_rule` parameters ([#544][2020-10-02_2])
- Add Zip functionality ([#539][2020-10-02_3])

**Bug Fixes:**

- Fix bug with updating a collaboration role to owner ([#536][2020-10-02_4])
- Allow `ints` to be passed in as item IDs ([#530][2020-10-02_5])
