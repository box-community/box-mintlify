## Box Content Preview v2.57.0 released

  **New Features and Enhancements:**

* Add react versions of core control components ([#1282][2020-11-13_1282])
* Add react versions of existing control icons ([#1280][2020-11-13_1280])
* Add react versions of fullscreen and zoom controls ([#1283][2020-11-13_1283])
* Add react versions of text viewer controls ([#1284][2020-11-13_1284])
* Add instrumentation for annotation creation ([#1275][2020-11-13_1275])
* Upgrade `react` and `react-dom` to `^16.9.0` ([#1278][2020-11-13_1278])

**Bug Fixes:**

* Select newly created annotation ([#1276][2020-11-13_1276])
* Allow download of files via embedded links in Chrome ([#1277][2020-11-13_1277])
* Revert `pdf.js` to `v2.2.228` due to font rendering issue ([#1274][2020-11-13_1274])
