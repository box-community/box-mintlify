## Box Java SDK v2.48.0 released

  `v2.48.0`

1. Add ability to get groups by name with fields option ([#789][2020-06-24_1])
2. Add shared link downscoping ([#785][2020-06-24_2])
3. Deprecate the use of float for Metadata values, in preference of the underlying value (double) ([#811][2020-06-24_3])
4. Add iterator support for group collaborations ([#813][2020-06-24_4])
5. Add ability to set the filename when uploading a new version of a file ([#810][2020-06-24_5])
6. Add support for the classification field for Files and Folders ([#809][2020-06-24_6])
7. Add support for setting Tracking Codes ([#766][2020-06-24_7])
8. Fix issue for `getIsExternallyOwned()` for Files and Folders ([#808][2020-06-24_8])
