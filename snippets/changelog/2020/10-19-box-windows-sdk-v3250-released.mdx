## Box Windows SDK v3.25.0 released

  ## 3.25.0 [2020-10-19]

- Add zip functionality ([#700][2020-10-19_2])
- Deprecate one of the overloaded `ExecuteMetadataQueryAsync()` methods ([#699][2020-10-19_3])
- Add support for `copyInstanceOnItemCopy` field for metadata templates  ([#698][2020-10-19_4])

**Bug Fixes:**

- Fix bug with JWT Authentication automatic retry ([#697][2020-10-19_5])
