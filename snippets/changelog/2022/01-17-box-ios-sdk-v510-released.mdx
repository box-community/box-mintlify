## Box iOS SDK v5.1.0 released

  ### New Features and Enhancements

* Allow to customize URL for the `OAuth2` authorization page ([#811][2022-01-17_1]) ([`1901d29`][2022-01-17_2])
* **`Events`:** Add support for `admin_logs_streaming stream` type ([#800][2022-01-17_3]) ([`0a3118e`][2022-01-17_4])
* **`RetentionPolicy`:** New API for get files and file versions under retention ([#807][2022-01-17_5]) ([`38327f0`][2022-01-17_6])
* **`SharedLink`:** add support for `vanity_name` ([#799][2022-01-17_7]) ([`3ea6eb2`][2022-01-17_8])

### Bug Fixes

* **`SignRequest`:** Fix encoding `date_value` to `yyyy-mm-dd` format in `prefillTag` ([#806][2022-01-17_9]) ([`4f902a4`][2022-01-17_10])
