## Box Java SDK v3.7.0 released

  ### New Features and Enhancements

* Add `is_accessible_via_shared_link` field to File and Folder ([#1103][2022-09-21_1]) ([`45e9906`][2022-09-21_2])

### Bug Fixes

* `BoxCollaboration.getItem()` returns `BoxItem.Info` not `BoxFolder.Info` ([#1102][2022-09-21_3]) ([`135850d`][2022-09-21_4]), closes [#1101][2022-09-21_5] [#1100][2022-09-21_6]. `BoxCollaboration.getItem()` used to return `BoxFolder.Info`. However, if collaboration was added on a file it would still return `BoxFolder.Info` which will end with throwing `BoxAPIException` when doing any API call. If you are getting collaboration item it is best to store it as `BoxItem.Info` or check its type and store it as `BoxFile.Info` or `BoxFolder.Info`.
* Add missing constructor to `BoxNotificationEmail` class ([#1098][2022-09-21_7]) ([`2534f34`][2022-09-21_8])
