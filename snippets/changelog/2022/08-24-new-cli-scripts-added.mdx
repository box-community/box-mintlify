## Updates to Box CLI

  The recent additions to [Box CLI script library][2022-08-24_1] and [CLI guides][2022-08-24_2]
improve automation even further and simplify handling repetitive tasks.

{/* more */}

## New CLI scripts

The sample scripts library now includes the following scripts for you to use:

* [Provision users and folders][2022-08-24_3] builds an
onboarding folder structure and adds users.
* [Manage groups and collaborations][2022-08-24_4] creates or updates groups, add users to them, and finally add groups to folders as collaborators.
* [Report inactive users][2022-08-24_5] generates a CSV file with a list of users who have been inactive for a specific number of days.
* [Update user zones][2022-08-24_6] provisions users to a specific 
data residency zone within a Multizone Box tenant.
* [Deprovision users][2022-08-24_7] removes users.
