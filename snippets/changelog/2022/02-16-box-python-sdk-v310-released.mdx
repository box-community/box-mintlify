## Box Python SDK v3.1.0 released

  ### New Features and Enhancements

* Add support for Python 3.10 ([#692][2022-02-16_1]) ([`d4aed82`][2022-02-16_2])
* Add support for Python 3.8, Python 3.9, `pypy-3.7` and `pypy-3.8`. ([#689][2022-02-16_3]) ([`0aa94cc`][2022-02-16_4])
* Deprecate `use_index` parameter from `MDQ` of files/folders ([#666][2022-02-16_5]) ([`2595720`][2022-02-16_6])
* Replace external package `mock` with Python standard library `unittest.mock` ([#697][2022-02-16_7]) ([`6fd6366`][2022-02-16_8])
* Upgrade cryptography library to the most recent version. ([#668][2022-02-16_9]) ([`9c94d08`][2022-02-16_10]), closes [#667][2022-02-16_11]

### Bug Fixes

* `UploadSession.commit` returns `None` when retry limit was reached ([#696][2022-02-16_12]) ([`9456fe0`][2022-02-16_13])
* Add missing `api_call` decorator for `create_upload_session` ([#686][2022-02-16_14]) ([`3510d3a`][2022-02-16_15])
* Fix chunked upload ([#673][2022-02-16_16]) ([`2605fd7`][2022-02-16_17]), closes [#671][2022-02-16_18]
