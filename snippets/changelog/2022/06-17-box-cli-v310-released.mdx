## Box CLI v3.1.0 released

  ### New Features and Enhancements

* Add --all flag for search to return all results ([#336][2022-06-17_1]) ([`23ea0a5`][2022-06-17_2])
* add unique state parameter to `OAuth2` login ([#292][2022-06-17_3]) ([`5ce6a40`][2022-06-17_4])
* allow changing base URLs ([#303][2022-06-17_5]) ([`e284059`][2022-06-17_6])
* obtain `oauth` authorization from command line ([#299][2022-06-17_7]) ([`18c88bb`][2022-06-17_8])
* use native credential storage for MacOS and Windows ([#295][2022-06-17_9]) ([`74c4922`][2022-06-17_10])
