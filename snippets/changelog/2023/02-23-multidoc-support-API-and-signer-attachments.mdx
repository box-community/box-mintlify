## Box Sign updates

  ## Multiple documents in one request

The Box Sign [multiple documents][2023-02-23_1] in one request is now
supported in our [public API][2023-02-23_2].

## Signer attachments

The [signer attachments][2023-02-23_3] feature allows users to request
additional files from signers in a file attachment field.

## Where to get support

Should you have any issues or need further guidance, please post a request to
our [developer forum][2023-02-23_4] for any help needed.
