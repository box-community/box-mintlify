## Improvements in App Access Level for JWT authentication

  You can now ensure access to enterprise Managed Users **and groups** that were not created by [apps authenticated with JWT][2023-07-26_1]. To do so, set **App Access Level** to **App + Enterprise Access** in the **Configuration** tab of the Developer console.
