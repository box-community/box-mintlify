## UI Elements in Box for Salesforce managed package

  With the [Box for Salesforce][2023-05-26_1] managed package you can use the following [UI Elements][2023-05-26_2] as Lightning Components: [content picker][2023-05-26_3], [explorer][2023-05-26_4], [preview][2023-05-26_5], and [uploader][2023-05-26_6]. You can use them on Lightning Pages, or in Flows.

## Where to get support

Should you have any issues or need further guidance, please post a request to our [developer forum][2023-05-26_7] for any help needed.
