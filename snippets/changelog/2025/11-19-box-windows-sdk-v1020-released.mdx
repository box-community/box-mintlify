## Box Windows SDK v10.2.0 released

  ### New Features and Enhancements

* Support Archive API (box/box-codegen[#865][2025-11-19_1]) ([#1266][2025-11-19_2]) ([`bca0473`][2025-11-19_3])
* Support GET enterprise configuration API (box/box-openapi[#559][2025-11-19_4]) ([#1198][2025-11-19_5]) ([`426aaed`][2025-11-19_6])
* Include `pdb` files when packing .net root project (box/box-codegen[#859][2025-11-19_7]) ([#1169][2025-11-19_8]) ([`8b7e7c8`][2025-11-19_9])
* Use generated `RetryStrategy` for retrying requests (box/box-codegen[#872][2025-11-19_10]) ([#1232][2025-11-19_11]) ([`5567fef`][2025-11-19_12])
