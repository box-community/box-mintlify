## Integration mappings for Teams

  Use the [integration mappings][2025-03-12_1] feature for MS Teams to manage [where the content
from Teams][2025-03-12_2] channels and chats is stored in Box.

* [List][2025-03-12_3] all Teams integration mappings
* [Create][2025-03-12_4] a new Teams integration mapping
* [Update][2025-03-12_5] existing Teams integration mapping
* [Delete][2025-03-12_6] Teams integration mapping

Check the integration mappings for Teams API developer [guide][2025-03-12_7] to learn more about this feature.

## Where to get support

Should you have any issues or need further guidance, please post a request to
our [developer forum][2025-03-12_8] for any help needed.
