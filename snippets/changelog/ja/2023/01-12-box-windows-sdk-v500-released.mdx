## Box Windows SDK v5.0.0のリリース

  ⚠ 重大な変更

* .NET Frameworkを4.6.2にアップグレード ([#881][2023-01-12_1])
* 非推奨となったメソッドを削除 ([#881][2023-01-12_1])
* `use_index`リファレンスを削除 ([#881][2023-01-12_1])
* `GetFileVersionsUnderRetentionForAssignmentAsync`から適切なオブジェクトが戻されるよう修正 ([#881][2023-01-12_1])

**新機能と機能強化:**

* .NET Frameworkを4.6.2にアップグレード ([#881][2023-01-12_1]) ([`f1989aa`][2023-01-12_2])、[#863][2023-01-12_3]をクローズ
* 非推奨となったメソッドを削除 ([#881][2023-01-12_1]) ([`f1989aa`][2023-01-12_2])、[#874][2023-01-12_4]をクローズ
* `use_index`リファレンスを削除 ([#881][2023-01-12_1]) ([`f1989aa`][2023-01-12_2])、[#870][2023-01-12_5]をクローズ

**バグ修正:**

* `IBoxFilesManager#ViewVersionsAsync`にページネーションオプションを追加 ([#869][2023-01-12_6]) ([`2324495`][2023-01-12_7])、[#866][2023-01-12_8]をクローズ
* `GetFileVersionsUnderRetentionForAssignmentAsync`から適切なオブジェクトが戻されるよう修正 ([#881][2023-01-12_1]) ([`f1989aa`][2023-01-12_2])、[#875][2023-01-12_9]をクローズ
