## Box Python SDK v3.1.0のリリース

  ### 新機能と機能強化

* Python 3.10のサポートを追加 ([#692][2022-02-16_1]) ([`d4aed82`][2022-02-16_2])
* Python 3.8、Python 3.9、`pypy-3.7`、および`pypy-3.8`のサポートを追加 ([#689][2022-02-16_3]) ([`0aa94cc`][2022-02-16_4])
* ファイル/フォルダの`MDQ`にある`use_index`パラメータを廃止 ([#666][2022-02-16_5]) ([`2595720`][2022-02-16_6])
* 外部パッケージの`mock`をPython標準ライブラリの`unittest.mock`に置き換え ([#697][2022-02-16_7]) ([`6fd6366`][2022-02-16_8])
* 暗号化技術ライブラリを最新バージョンにアップグレード ([#668][2022-02-16_9]) ([`9c94d08`][2022-02-16_10])、[#667][2022-02-16_11]をクローズ

### バグ修正

* 再試行の上限に達したときに`UploadSession.commit`によって`None`が返される ([#696][2022-02-16_12]) ([`9456fe0`][2022-02-16_13])
* `create_upload_session`に欠落している`api_call`デコレータを追加 ([#686][2022-02-16_14]) ([`3510d3a`][2022-02-16_15])
* 分割アップロードを修正 ([#673][2022-02-16_16]) ([`2605fd7`][2022-02-16_17])、[#671][2022-02-16_18]をクローズ
