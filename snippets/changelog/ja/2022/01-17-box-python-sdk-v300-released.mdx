## Box Python SDK v3.0.0のリリース

  **重大な変更**

* Python 2.7のサポートを終了 ([#645][2022-01-17_1])
* 不足しているパラメータ`stream_position to get_admin_events`メソッドを追加 ([#648][2022-01-17_2])
* Python 3.5のサポートを終了 ([#654][2022-01-17_3])
* 不適切な用語を使用している非推奨のコードを削除 ([#651][2022-01-17_4])
* 一部の関数でキーワードのみの引数の使用を強制 ([#656][2022-01-17_5])

**新機能と機能強化:**

* `six`ライブラリと`__future__` importを削除 ([#646][2022-01-17_6])
* メソッドのパラメータにタイプヒントを追加 ([#650][2022-01-17_7])

**バグ修正:**

* `multi-input`の呼び出しに不足している`api_call`デコレータを追加 ([#653][2022-01-17_8])
* `mypy`がタイプヒントを認識できるように`py.typed`ファイルを追加 ([#657][2022-01-17_9])
