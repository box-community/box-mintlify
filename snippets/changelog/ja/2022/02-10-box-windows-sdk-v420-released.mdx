## Box Windows SDK v4.2.0のリリース

  **バグ修正:**

* 企業からユーザーをロールアウトしたときに正しくnullを渡す ([#792][2022-02-10_1]) ([`c85c573`][2022-02-10_2])
* `BoxAuthenticationFailedException`の作成により例外がスローされなくなる ([#790][2022-02-10_3]) ([`55a706e`][2022-02-10_4])
* `AutoPaginate`のnull引数の例外 ([#666][2022-02-10_5]) ([`c61f08c`][2022-02-10_6])

**新機能と機能強化:**

* `admin_logs_streaming`のサポートを追加 ([#797][2022-02-10_7]) ([`a775e1e`][2022-02-10_8])
* クライアント資格情報許可の`auth`のサポートを追加 ([#799][2022-02-10_9]) ([`b8a64ca`][2022-02-10_10])
* `disposition_at`フィールドをファイルオブジェクトに追加 ([#793][2022-02-10_11]) ([`2766a91`][2022-02-10_12])
* `BoxConfig`で`auth`トークンの`uri`を設定する可能性を追加 ([#794][2022-02-10_13]) ([`ae8cd8b`][2022-02-10_14])
* `ExecuteMetadataQuery`の`index_name`を廃止 ([#800][2022-02-10_15]) ([`6a6a0e4`][2022-02-10_16])
