## Box Java SDK v2.48.0のリリース

  `v2.48.0`

1. フィールドオプションにより名前ごとにグループを取得する機能を追加 ([#789][2020-06-24_1])
2. 共有リンクのダウンスコープを追加 ([#785][2020-06-24_2])
3. メタデータ値の浮動小数点の使用を非推奨にして、基になる値 (double) を優先 ([#811][2020-06-24_3])
4. グループコラボレーション用に反復子のサポートを追加 ([#813][2020-06-24_4])
5. ファイルの新しいバージョンをアップロードする際にファイル名を設定する機能を追加 ([#810][2020-06-24_5])
6. ファイルとフォルダの分類フィールドのサポートを追加 ([#809][2020-06-24_6])
7. 追跡コードの設定のサポートを追加 ([#766][2020-06-24_7])
8. ファイルとフォルダの`getIsExternallyOwned()`に関する問題を修正 ([#808][2020-06-24_8])
