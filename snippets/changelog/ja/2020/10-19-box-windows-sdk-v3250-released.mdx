## Box Windows SDK v3.25.0のリリース

  ## 3.25.0 [2020年10月19日][2020-10-19]

**新機能と機能強化:**

* グループ取得時のフィルタのサポートを追加 ([#703][2020-10-19_1])
* 圧縮機能を追加 ([#700][2020-10-19_2])
* オーバーロードされた`ExecuteMetadataQueryAsync()`メソッドの1つを廃止 ([#699][2020-10-19_3])
* メタデータテンプレートの`copyInstanceOnItemCopy`フィールドのサポートを追加 ([#698][2020-10-19_4])

**バグ修正:**

* JWT認証の自動再試行に関するバグを修正 ([#697][2020-10-19_5])
