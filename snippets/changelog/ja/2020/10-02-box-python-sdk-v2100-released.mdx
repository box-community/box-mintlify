## Box Python SDK v2.10.0のリリース

  **新機能と機能強化:**

* メタデータテンプレートの`copyInstanceOnItemCopy`フィールドのサポートを追加 ([#546][2020-10-02_1])
* `action`パラメータと`completion_rule`パラメータを使用したタスクの作成を許可 ([#544][2020-10-02_2])
* 圧縮機能を追加 ([#539][2020-10-02_3])

**バグ修正:**

* コラボレーションロールを所有者に更新する際のバグを修正 ([#536][2020-10-02_4])
* `ints`を項目IDとして渡すことを許可 ([#530][2020-10-02_5])
