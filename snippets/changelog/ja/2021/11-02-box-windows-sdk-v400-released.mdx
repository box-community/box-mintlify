## Box Windows SDK v4.0.0のリリース

  **重要な変更:**

* BoxClientおよびマネージャのインターフェースを抽出し、テスタビリティを改善 ([#603][2021-11-02_1])
* BoxConfigBuilderを追加し、BoxConfigを変更不可に設定 ([#737][2021-11-02_2])
* 非同期メソッドからタスクを公開 ([#742][2021-11-02_3])
* DateTimeの代わりにDateTimeOffsetを使用 ([#749][2021-11-02_4])
* 返された例外を修正 ([#753][2021-11-02_5])
* .NET Standardを2.0にアップグレード ([#755][2021-11-02_6])

**新機能と機能強化:**

* 割り当てのリテンションの対象となるファイルおよびファイルバージョンを取得する機能を追加 ([#734][2021-11-02_7])
* `Folder`の更新のために`is_collaboration_restricted_to_enterprise`フラグのサポートを追加 ([#732][2021-11-02_8])
* 不適切な用語の変更 ([#738][2021-11-02_9])
* サービス利用規約のユーザーステータスの作成用に使いやすくなった新しいメソッドを追加 ([#740][2021-11-02_10])
* ごみ箱内の項目を取得する際に並べ替えと方向のパラメータを渡すことを許可 ([#754][2021-11-02_11])
* タスクのcompletion_ruleフィールドのサポートを追加 ([#758][2021-11-02_12])
* BoxSign APIのサポートを追加 ([#765][2021-11-02_13])

**バグ修正:**

* アップロード時の`Cannot access a closed Stream.Request`例外を修正 ([#739][2021-11-02_14]) ([#757][2021-11-02_15])

- https://www.nuget.org/packages/Box.V2/4.0.0>

- https://www.nuget.org/packages/Box.V2.Core/4.0.0>
