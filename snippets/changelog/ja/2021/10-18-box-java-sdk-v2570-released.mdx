## Box Java SDK v2.57.0のリリース

  **新機能と機能強化:**

* `BoxFolder.getChildren`でのマーカーベースのページ割りのサポートを追加 ([#927][2021-10-18_1])
* `minimal-json`をバージョン0.9.5にアップグレード
* `jose4j`をバージョン0.7.9にアップグレード
* バージョン4.0.1にGradleラッパーを追加 ([#928][2021-10-18_2])

**バグ修正:**

* 無限再帰の修正 ([#924][2021-10-18_3])
* `BoxFile`および`BoxFolder`の`BoxSharedLink`にバニティURLを設定できないバグの修正 ([#925][2021-10-18_4])
