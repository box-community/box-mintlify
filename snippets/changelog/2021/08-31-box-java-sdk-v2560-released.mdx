## Box Java SDK v2.56.0 released

  **New Features and Enhancements:**

* Replace `submaster` `GroupMembershipRole` with `coadmin`. Replace `MASTER_INVITE_ACCEPT` and `MASTER_INVITE_REJECT` with `ADMIN_INVITE_ACCEPT` and `ADMIN_INVITE_REJECT`. ([#907][2021-08-31_1])
* Add `tracking_codes` to create User API call ([#910][2021-08-31_2])

**Bug Fixes:**

* Fix `url` for `BoxFileRequest.Info` object ([#906][2021-08-31_3])
* Attempt to fix thread locking issue on refresh of access token ([#912][2021-08-31_4])
