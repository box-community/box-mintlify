## Box Windows SDK v4.0.0 released

  **Breaking changes:**

* Extract interfaces for BoxClient and Managers to improve testability ([#603][2021-11-02_1])
* Add BoxConfigBuilder and make BoxConfig immutable ([#737][2021-11-02_2])
* Expose tasks from async methods ([#742][2021-11-02_3])
* Use DateTimeOffset instead of DateTime ([#749][2021-11-02_4])
* Rework returned exceptions ([#753][2021-11-02_5])
* Upgrade .NET Standard to 2.0 ([#755][2021-11-02_6])

**New Features and Enhancements:**

* Add ability to get files under retention for assignment and file versions under retention for assignment ([#734][2021-11-02_7])
* Add `is_collaboration_restricted_to_enterprise` flag support for `Folder` update ([#732][2021-11-02_8])
* Replace insensitive language ([#738][2021-11-02_9])
* Add new, easier to use method for create terms of service user status ([#740][2021-11-02_10])
* Allow sort and direction parameter to be passed in when getting trashed items ([#754][2021-11-02_11])
* Add support for Task completion_rule field ([#758][2021-11-02_12])
* Add BoxSign API support ([#765][2021-11-02_13])

**Bug Fixes:**

* Fix `Cannot access a closed Stream.Request` exception during upload ([#739][2021-11-02_14]) ([#757][2021-11-02_15])

- https://www.nuget.org/packages/Box.V2.Core/4.0.0
