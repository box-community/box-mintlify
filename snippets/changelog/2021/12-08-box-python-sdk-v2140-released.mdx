## Box Python SDK v2.14.0 released

  **New Features and Enhancements:**

* Add `admin_logs_streaming` support for events stream ([#623][2021-12-08_1])
* Add `vanity_name` parameter for creating shared link to a file or folder ([#637][2021-12-08_2])
* Add getting files and file versions under retention for a retention policy assignment ([#633][2021-12-08_3])
* Support base item operations for WebLink class ([#639][2021-12-08_4])

**Bug Fixes:**

* Limit cryptography to version \<3.5.0 ([#636][2021-12-08_5])
* Avoid raising 404 when a thumbnail cannot be generated for a file ([#642][2021-12-08_6])
