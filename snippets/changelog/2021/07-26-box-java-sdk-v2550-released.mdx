## Box Java SDK v2.55.0 released

  NOTE: Due to the benign nature of the "breaking change" below, we decided NOT to increment the major version for this release.  There should be no customer impact due to this change.

**Breaking Changes:**

* Update execute metadata query to match API response ([#890][2021-07-26_1])
  * NOTE: This change removes a method without deprecating it.  It was not possible to use the method correctly at all, because the underlying service no longer supported it.

**New Features and Enhancements:**

* Remove or deprecate insensitive language (\[[#889][2021-07-26_2]])(https://github.com/box/box-java-sdk/pull/889)
* Add support for `is_external_collab_restricted` parameter for User ([#896][2021-07-26_3])
* Add configurable permissions support for `GroupMembership` ([#897][2021-07-26_4])
* Add `SHIELD_JUSTIFICATION_APPROVAL` event type ([#898][2021-07-26_5])
* Add ability to get files under retention for assignment and file versions under retention for assignment ([#899][2021-07-26_6])
* Add `TASK_UPDATE`, `FILE_VERSION_RESTORE` and `ADVANCED_FOLDER_SETTINGS_UPDATE` event types ([#902][2021-07-26_7])
* Add Box Sign API support ([#904][2021-07-26_8])

**Bug Fixes:**

* Add setters for `BoxLegalHoldPolicy` ([#885][2021-07-26_9])
* Add setters for `BoxTaskAssignment` ([#886][2021-07-26_10])
* Add setters for Group Membership and Web Links ([#887][2021-07-26_11])
* Add setters for Webhooks ([#888][2021-07-26_12])
* Deprecate `BoxFile.getThumbnail` in favor of `BoxFile.getRepresentationContent` ([#891][2021-07-26_13])
