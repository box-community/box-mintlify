## Box iOS SDK v4.4.0 released

  **Breaking Changes:**

**New Features and Enhancements:**

* Add support for search parameter to get shared link items ([#756][2021-04-20_1])
* Add support for folder lock functionality ([#759][2021-04-20_2])
* Add support for `copyInstanceOnItemCopy` field for metadata templates ([#763][2021-04-20_3])
* Add support for stream upload of new file versions and add support for 'If-Match' header when uploading new file versions ([#766][2021-04-20_4])
* Add additional details field for `Event` model ([#770][2021-04-20_5])

**Bug Fixes:**

* Pass only a scheme to iOS Authentication APIs ([#755][2021-04-20_6])
* Update `listEnterpriseGroups()` to use documented parameter for filtering by name ([#757][2021-04-20_7])
* Fix bug for OAuth where the callback is not called if token has been revoked ([#762][2021-04-20_8])
