## Box Node SDK v1.38.0 released

  **New Features and Enhancements:**

* Add sensitive language event types for admin invites ([#648][2021-08-06_1])
* Use `BetterDocs` to adapt `JSDocs` to `TypeScript` ([#646][2021-08-06_2])
* Change `ProxyAgent` import to be dynamic ([#641][2021-08-06_3])
* New API for get files and file versions under retention ([#585][2021-08-06_4])

**Bug Fixes:**

* Deeply freeze Config except Buffers and Readable streams ([#651][2021-08-06_5])
* Fix a typo in docs of `src/managers/search.ts` ([#649][2021-08-06_6])
* Update broken documentation link ([#647][2021-08-06_7])
* fix type annotations for `exchangeToken` functions ([#645][2021-08-06_8])
* Deprecate files `getThumbnail` API in favor of 
  `getRepresentationContent` ([#627][2021-08-06_9])
