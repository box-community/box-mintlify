| Endpoint                                     | Path                     |
| -------------------------------------------- | ------------------------ |
| [Create a folder](/reference/post-folders)          | `POST /folders`    |
| [Upload a file](/reference/post-files-content)      | `POST /files/content`    |
| [Download a file](/reference/get-files-id-content)  | `GET /files/:id/content` |
| [Delete a file](/reference/delete-files-id) | `DELETE /files/:id`      |
| [Get embed link](/reference/get-files-id)          | `GET /files/:id`         |
